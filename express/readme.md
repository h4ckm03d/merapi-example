# Hello World

## Prerequisite

- nodejs
- typescript `$ yarn add global typescript`

## Development

- `$ yarn`
- `$ tsc` 
- `$ node start`
- output will be
```
[INFO] express_sample/app 2017-09-05 14:12:53 : Starting express on localhost:8090
[INFO] express_sample/service 2017-09-05 14:12:53 : express_sample@1.0.0 service running on 0.0.0.0:5000 in development mode
```
