# Merapi Example

## What is [Merapi](https://github.com/kata-ai/merapi)? 

  Pluginable Dependency Injection Container for Node.js

## Prerequisites
- Nodejs
- Typescript

## Setup
```
npm install -g typescript

npm install // or yarn install ( install yarn first!)

```

## List Example

- [Hello world](https://github.com/h4ckm03d/merapi-example/tree/master/hello-world)
```
// yarn
yarn hello-world

// npm
npm run hello-world
```
- [Hello world with Loggly](https://github.com/h4ckm03d/merapi-example/tree/master/hello-world-loggly)
```
// yarn
yarn hello-world-loggly

// npm
npm run hello-world-loggly
```
- [plugin Express example](https://github.com/h4ckm03d/merapi-example/tree/master/express)
```
// yarn
yarn express

// npm
npm run express
```
